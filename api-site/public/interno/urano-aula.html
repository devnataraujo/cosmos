Urano e os vetores

Nessa nova missão, vamos precisar levar muitas coisas, mas antes disso vamos conhecer os primos das variáveis, os arrays. Ele são como elas mas deixam você levar um monte de coisa diferente (ou igual) num lugar só.

Como por exemplo:

var caixaUm =

“Banana”

var caixaDois =

“Maçã”

Então quando você precisar pegar as frutas, não vai precisar guardar cada uma em uma caixa, você pode guardar todas na mesma caixa se usar o vetor

Assim como nas variaveis, você tambem pode colocar coisas diferentes, sejam numeros, palavras ou os nossos objetos da missão. Como por exemplo:

var endereço = [
“Rua dos Aliens”,
125,
“Planeta dos Aliens”
];
