Netuno e as variaveis

A primeira coisa que precisamos aprender antes de sairmos de Netuno é sobre as variáveis. Variaveis são caixinhas que guardam coisas. Essas caixinhas são pequenas e só deixam guardam uma unica coisa. Você pode usar assim:

var caixaUm = “Banana”

var caixaDois = “Maçã”

Quando você precisar da Banana é so você pegar a caixaUm
Quando você precisar da Maçã é só pegar a caixaDois

Outra coisa legal para saber sobre as variaveis é que elas podem guardar coisas diferentes como numeros e palavras, e no nosso caso os objetos da missão. Nós tambem podemos dar o nome que quisermos para as variaveis

var Foguete = “Combustivel”

var Idade = 5
